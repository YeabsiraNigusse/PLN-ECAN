# ECAN-PLN Integration Experiment

This Experiment demonstrates the integration of **ECAN (Economic Cognitive Attention Network)** with **PLN (Probabilistic Logic Network)** using Hyperon MeTTa.

## Overview

The integration uses ECAN as a "cognitive filter" to manage computational resources for logical reasoning. In complex knowledge bases, PLN can suffer from combinatorial explosion. ECAN mitigates this by identifying and prioritizing the most relevant facts (atoms) based on current context (stimulation), creating an **Attentional Focus (AF)** that serves as the input for PLN.
